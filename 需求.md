# 英语学习 App 项目需求与开发指南

## 一、 产品核心逻辑定义 (Core Logic)

1. **滑动自选分级 (Swipe-to-Select Onboarding)**：系统预设从 **0基础 (Level 0)** 到 **流利 (Level N)** 的多套固定难度段落。用户通过左右滑动卡片自由预览，听到/看到觉得适合自己当前水平的段落，直接确认为初始等级。
2. **0基础兜底 (Zero-Basis Support)**：特别兼顾完全不懂英语的用户。0基础的课文将从最简单的字母发音、打招呼（如 "Hello", "I am..."）和基础词汇开始，并配以极强的中文引导。
3. **N+1 学习法**：每次生成的课文在用户当前水平上，引入适量新单词和下一个语法框架，保持挑战性。
4. **主课文学习流**：
   - 第一阶段【磨耳朵】：先听，结合翻译和生词本理解。
   - 第二阶段【开口跟读】：看着课文朗读，调用语音评测 API，**>= 60分及格通过**。
   - 第三阶段【互动问答】：AI 出3道题，LLM 判定回答对错。
5. **遗忘曲线复习**：按 1, 2, 3, 5, 7, 15, 30 天复习课文。复习要求：**完整朗读课文并达到 60分及格**。
6. **AI 助教与订阅制**：全局悬浮 AI 答疑机器人；采用免费试用+订阅制商业模式。

## 二、 用户交互指南 (User Interaction Guide)

### 1. 引导与定级流程 (Onboarding Flow)

- **View 1: 目标收集** -> 选择学习目标（日常、出国、职场、考试等）。
- **View 2: 难度滑动选择 (Level Selector Swiper)** ->
  - 界面呈现一个大卡片轮播图 (Carousel/Swiper)，包含 Level 0 到 Level N（如新概念4难度）。
  - **Level 0 卡片**：明确标注“完全零基础”，展示最基础的内容（如 "Hi, I am Tom."），配有中英双语和音频。
  - **Level 1-N 卡片**：展示对应难度的固定段落，带音频播放按钮。
  - 交互：用户左右滑动翻动等级卡片，找到刚好能听懂或稍微吃力的那一张，点击“就从这里开始” (Select this Level)。
- **View 3: 订阅引导 (Paywall)** -> 提示新用户获赠试用期，展示 VIP 订阅计划。

### 2. 主页与仪表盘 (Dashboard)

- **View 1: 学习日历** -> 顶部展示打卡记录。
- **View 2: 今日任务** ->
  - **「今日新课」**：进入 N+1 动态生成课文。
  - **「待复习课文」**：列表展示遗忘曲线触发的复习任务。

### 3. 每日学习主界面 (Daily Learning Space)

- **全局组件：悬浮 AI 助教 (Floating AI Tutor)** -> 点击呼出 Chat 界面，基于当前课文上下文解答疑惑（支持语音/文字提问）。
- **Step 1: 磨耳朵 (Listening)** -> 播放 AI 音频，侧边/底部提供翻译、生词、语法点 Tabs。
- **Step 2: 朗读跟读 (Shadowing)** -> 用户按住录音，微软发音评估返回分数，**>= 60 分标绿通过**。
- **Step 3: 课后问答 (Q&A)** -> AI 出 3 题，用户回答，LLM 判定。
- **Step 4: 结算页 (Completion)** -> 弹出庆祝卡片并保存笔记。

### 4. 复习模式 (Review Mode)

- **交互流**：点击待复习课文 -> 听一遍（可选） -> **全文朗读录音** -> 评分 >= 60 分 -> 标记复习完成。

### 5. 周边模块 (Modules)

- **单词本、错题本、笔记、游乐园 (Playground)** 保持原定聚合与游戏化逻辑。
